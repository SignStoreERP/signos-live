Standard Operating Procedure

← Back to Hub
1

Size and Quantity

Input the width and height of the finished product. Set the quantity. The system instantly begins mapping the order to the internal Master Retail Curve.

2

Define Variables

Select options such as material thickness, sides, or custom cuts. These selections trigger background labor multipliers (e.g. CNC setup fees or contour weeding adders).

3

Check Retail Price

Review the calculated total in the blue Results panel. This is your market-competitive price. Note that quantity discounts apply automatically as you increase volume.

4

Sync with Workflow

The system auto-generates a formatted job description at the bottom of the page. Copy and paste this directly into QuickBooks or Freshdesk to maintain data integrity for the production team.
